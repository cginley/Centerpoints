Centerpoints
Number of points:

Given a set of d-dimensional points, a point is called a centerpoint if any hyperplane passing through this point has at least 1/(d + 1) points on either side of it.

A point p is considered a centerpoint of a set of 1D points S if at least 1/2 of the points fall on either side of p. This is commonly referred to as the median of S.

A point p is considered a centerpoint of a set of 2D points S if at least 1/3rd of the point fall on either side of l, where l is any line that passes through p. In most cases, there is not one unique centerpoint. A set of points has an area that contains all of its centerpoints.
